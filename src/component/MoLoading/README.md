## MoLoading使用文档


##### 提供的配置接口

参数|功能|默认值|其他
:-----:|:-----:|:-----:|:-----:
size|大小|'default'|提供三种大小，'small','default','large'
text|动画下方文字|''|默认不显示
color|动画和文字颜色|'#1890ff'|无
style|动画风格|'spinDot'|提供多种风格,'spinDot','jumpBlock','spinBlock','switchDot','slipBlock','slipDot','blinkDot','spreadDot','roundDot',
